# NUXT Custom Social Login 

##### Pre Configuration

Add the google oAuth client ID, Facebook app client ID and the API base url first

##### Run the project in dev mode
````
npm install
npm run dev
````

##### Run the project in production mode
````
npm install
npm run start
````

* Port = 3000
* Host = 0.0.0.0
